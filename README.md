# 🏛️ CIVILISATION — Le Jeu de la Famille Dardeau (v0.1)

Jeu voxel **indépendant** (aucun Minecraft requis) : un seul fichier
`civilisation.html` contenant tout le moteur. Age of Empires × Il était une
fois... l'Homme × le dossier Civilisation.

## Jouer

- **PC** : double-clic sur `JOUER_CIVILISATION.bat` (plein écran)
- **Téléphone / tablette** : ouvrir l'adresse du jeu en ligne (voir ci-dessous)
  → joysticks tactiles automatiques + boutons ⛏️ 🧱 ⬆️
- **Vocal** 🎤 : bouton micro en haut à droite, puis dire :
  *menu, craft, inventaire, maestro, aide, saute, ferme*
- Le **Maestro parle** (voix douce française) : consignes orales, répétables.

## Contenu v0.1

- Monde voxel généré : plaine, forêt, désert, neige, océan, volcan, arbres, fleurs
- Casser/poser des blocs, inventaire, fabrication (planches, torches, feu de camp, peintures)
- 4 profils (Léana 🌸, Mathias 🔨, Papa 👑, Invité) — quêtes adaptées à chacun
- 8 avatars héros (Pharaonne, Gladiateur, Samouraï, Viking, Inventeur·e, Astronome, Super-Héros, Guerrière)
- 8 difficultés : baby → hardcore
- Ère 1 **Préhistoire** : 3 quêtes par profil, racontées vocalement, confettis et fanfare
- Menu **Modes** extensible : Histoire/Quêtes, Construction libre, Survie actifs —
  Archéologie, Astronomie, Chimie, Maths, Mythologie, Guerre, Colonie,
  Technologie, Musique, Inventions = cartes « greffables » (le plan : un mode = un module)
- Sauvegarde automatique par profil (sur l'appareil)
- Adapté TDAH : objectif toujours affiché, une consigne à la fois, répétition
  à volonté, récompenses immédiates, aucune punition

## Roadmap

v0.2 : Néolithique + Antiquité, PNJ villageois, multijoueur local (écran d'un
appareil chacun via le site famille), premiers modes greffés (Astronomie pour
Mathias, Musique pour Léana). v0.3 : colonie (bâtiments à construire par plans),
guerre/stratégie AoE. Le jeu est la **base** : chaque nouveau projet devient un
mode du menu.
